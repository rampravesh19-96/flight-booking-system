# Flight Booking and Confirmation System

A portfolio-grade travel tech application scaffold for a flight search and booking experience.

## Milestone 1
- Project scaffolding and base setup
- Frontend: Next.js + TypeScript + Tailwind CSS
- Backend: Express.js + TypeScript
- Prisma setup for MySQL-compatible database
- Basic health route
- Landing page

## Repo Structure
- `frontend/` — Next.js app
- `backend/` — Express API
- `prisma/` — Prisma schema and seed setup

## Run locally
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start frontend:
   ```bash
   npm run dev:frontend
   ```
3. Start backend:
   ```bash
   npm run dev:backend
   ```

## Verify
- Frontend: visit `http://localhost:3000`
- Backend: visit `http://localhost:4000/api/health`
